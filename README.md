## DNS Resolver
This is a DNS Resolver implementation.

## Why
I use a lot of AI at work, so I wanted to build something the "old fashioned" way where I can write code, make syntax errors, debug, and repeat until something works. I want the feeling of building something from scratch and struggling again. 

I still like using AI and I definitely think there's a middle ground, but I just kind of want to make something on my own.

Kind of... I'm writing code myself but I'm still learning through online resources including AI. I hope that writing the code myself will stretch a muscle in my mind that hasn't been stretched in a while.

## Why DNS
It was one of the first things I thought of. It seems pretty technical and non-trivial. I learned about DNS in university and have an understanding of the purpose it serves, but I don't know how it actually works behind the scenes. To start, I wrote about it here https://meshan.dev/blog/coffee-codex-dns-resolver/.

## Why Rust
No real reason, just wanted to use it since I haven't in a while and I want to go to the [Seattle Rust User Group](https://www.meetup.com/seattle-rust-user-group/) meetup in Bellevue and have something to talk about.
